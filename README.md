# 🌿 EcoShop – Sustainable E-Commerce Platform

EcoShop is a full-stack sustainable e-commerce web application engineered to integrate carbon footprint awareness into the online shopping experience.  
The platform enables environmentally conscious purchasing decisions through carbon emission tracking and eco-product insights.

---

## 🚀 Technology Stack

### Backend
- Java 17
- Spring Boot
- Spring Security
- JWT Authentication
- Hibernate / JPA
- Maven

### Frontend
- React (Vite)
- Axios
- Responsive UI Components

### Database
- MySQL 8

---

## 🔐 Core Features

### Authentication & Security
- Secure user registration and login
- JWT-based authentication
- Role-based authorization (User / Admin / Seller)
- Protected REST APIs

### Product Management
- Product CRUD operations
- Product listing with:
  - Image
  - Price
  - Carbon Footprint (CO₂e/kg)
  - Eco Badge
- Search and filtering capabilities
- Detailed product view

### Smart Cart System
- Add/remove products
- Quantity management
- Dynamic price calculation
- Real-time carbon emissions summary
- Order checkout workflow

---
## 🗄 Database Configuration

Database: `eco_store`  
Engine: MySQL 8  
Schema generation: Hibernate (ddl-auto=update)
